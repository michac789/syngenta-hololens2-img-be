Syngenta Hololens2 Backend
View Admin Page
This backend server host the PDF images required for Syngenta Hololens2 application. Refer to the instruction below on how to use.
  • Log in to Admin Page (link above) with the credentials (username: superuser, password: syngenta_ar_2024).
  • Click on any of the 7 stations name to access it (MS1, MS2, MS3, MS4, MS5, MS6, WH).
  • You can add new image or edit existing one. There are 3 fields for each entry: label, order, image; label is an optional string to label an image, order should be unique integer from 1-100 which determines the order in which the images are returned (ascending order), image is simply the image itself.

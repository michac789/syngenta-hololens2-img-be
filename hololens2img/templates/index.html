Syngenta Hololens2 Backend
View Admin Page
This backend server host the PDF images required for Syngenta Hololens2 application. Images for each station are stored in a different model. Refer to the instruction below on how to modify existing image.
  • Log in to Admin Page (link above) with your credentials, or create a new super user by executing `python manage.py createsuperuser`. Make sure that you are in the correct directory where `manage.py` is located.
  • There are 7 stations as follows: MS1, MS2, MS3, MS4, MS5, MS6, WH, each has a different model. In admin page under 'HOLOLENS2IMG', click on a station to access it.
